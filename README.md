## Verify Your Download

Every release ships with a `SHA256SUMS.txt` file. **Always verify before installing.**

**macOS / Linux**

```bash
shasum -a 256 UKey-Wallet-1.1.2-mac-arm64.dmg
```

**Windows (PowerShell)**

```powershell
Get-FileHash UKey-Wallet-1.1.2-win-x64.exe -Algorithm SHA256
```

The output must match the value listed in `SHA256SUMS.txt` of the same release. If they do not match, do not install — the file may have been tampered with.

## Features

**Multi-chain.** 40+ blockchains out of the box — EVM, Bitcoin, Solana, TON, Cosmos, Polkadot, Sui, Aptos, Cardano, Tron, Near, Algorand, and more.

**Hardware integration.** Pairs with the UKey hardware family — Core 26, Lite Card, Seed Ring, Seed Ti — for air-gapped signing, NFC backup and PIN protection.

**DApp connectivity.** WalletConnect v2, built-in DApp browser, and inpage providers for EVM, Solana, Sui and others.

**Asset management.** Multi-account portfolio, transaction history, NFT gallery, custom tokens and networks.

**Markets.** Real-time prices, charts, watchlists and token discovery.

## Security

- AES-256 local encryption for all sensitive storage
- Hardware-isolated signing — keys never leave the device
- Transaction risk detection (phishing addresses, suspicious contracts)
- Biometric unlock (Face ID, Touch ID, Fingerprint)
- Optional encrypted cloud backup

Found a security issue? Email **[support@ukey.com](mailto:support@ukey.com)** with details. Do not open a public issue.
